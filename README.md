# 生成 AI の取扱説明書

## プロジェクト概要

「生成 AI の取扱説明書」は、50 代以上の方（親世代）をターゲットにした、生成 AI について優しく学べる Web ポータルサイトです。
「新しい家電の取扱説明書」のように、専門用語を避け、親しみやすいデザインとわかりやすい言葉で、AI の基本や活用方法を紹介します。

## 主な機能・特徴

- **親しみやすい UI**: 優しい色使い、大きな文字、カード形式のレイアウトを採用。
- **対話形式の解説**: 案内係（AI キャラクター）が会話形式で分かりやすく説明します。
- **専門用語の「翻訳」**: 「プロンプト」→「お願いメモ」のように、日常の言葉に言い換えて解説。
- **追従型タブナビゲーション**: 縦長のスマホ画面でも、常に上部にメニューが表示され、迷わずページ移動が可能。
- **活用例の絞り込み**: 「家事」「旅行」などのジャンルボタンで、自分に関係ある活用例だけを簡単に見つけられます。
- **便利なAIツール紹介**: 初心者にも使いやすい、おすすめの生成AIツールを厳選して紹介。
- **用語集**: 難しい言葉にはツールチップで解説を表示し、その場ですぐに意味を確認できます。
- **シングルファイル配布**: ビルド後は1つのHTMLファイルになるため、サーバー設定なしで簡単に共有・閲覧が可能です。

## 開発環境

- React
- TypeScript
- Vite
- TailwindCSS (Play CDN)
- React Router
- Lucide React (アイコン)
- vite-plugin-singlefile (シングルファイルビルド)

## 始め方 (開発者向け)

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

## ディレクトリ構成

- `src/components`: UI パーツ（カード、吹き出し、Q&A など）
- `src/layout`: ヘッダー、フッター、ナビゲーションなどのレイアウト
- `src/pages`: 各ページ（ホーム、基本、活用例、ツール）
- `src/data`: 表示データ（ツール情報、活用例など）
